Verhalenwebsite

2019 | Responsive website | Frontend Development



De opdracht

Bij dit vak was het de bedoeling om een responsive website te maken, een website die zich automatisch aanpast aan de context, het apparaat, de omgeving en de interactie mogelijkheden. Dit vak was onderdeel van project Web, een project dat bestond uit de vakken: Vormgeving, Design Patterns en Frontend Development. Het hoofddoel van project Web was om een verhalenwebsite voor een reismaatschappij te maken. De maatschappij die mij werd aangewezen was Eurostar.


Wat heb ik geleerd?

Ik heb geleerd dat je met media queries, het uiterlijk kunt aanpassen per formaat en dat pixels niet meeschalen dus dat je beter em, rem om procenten kunt gebruiken in je code. Het gebruik van flexbox en grid was nieuw voor mij en heeft me geholpen bij het ordenen van de content. Bij mobiel formaat heb ik 1 kolom gebruikt maar op groter formaat gebruik ik een grid met meerdere kolommen. Ik heb het laadbalkje (een microanimatie) gemaakt met behulp van JavaScript en verder nog wat css transities gebruikt.
